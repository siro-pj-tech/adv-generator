# AI ADV プロンプトジェネレーター v1.0

ChatGPTやGeminiなどの対話型AIを、瞬時に熟練の「ゲームマスター（GM）」へと変貌させるプロンプト作成ツールです。画像生成と物語進行を2ステップに分けることで、AIの動作を安定させ、没入感のあるアドベンチャーゲーム体験を提供します。

## 🌟 特徴

- 2ステップ・プロンプト方式: 画像生成とゲーム開始を分離することで、ChatGPTでの「画像内文字化け」や「生成停止」を回避。

- 多彩なプリセット: 推理サスペンス、ダンジョンRPG、脱出ゲームの3つの王道シナリオを搭載。

- カスタムアドベンチャー: ジャンル、世界観、勝利条件、管理パラメータ（HP、時間など）を自由に設定可能。

- 画風カスタマイズ: リアル、アニメ、水彩、油絵など、AI画像生成のスタイルを指定可能。

- レスポンシブデザイン: PCでもスマートフォンでも快適にプロンプトを生成できます。

## 🚀 使い方

1. **設定の入力**

    - シナリオを選択し、プレイヤー名や難易度を設定します。

    - お好みのビジュアル画風を選択します。

2. **プロンプトの生成**

    - 「プロンプトを生成する」ボタンをクリックします。

3. **Step 1: ビジュアル生成**

    - 「ビジュアル生成用プロンプト」をコピーして、AI（ChatGPT/Gemini）に送信します。

    - AIがゲームの世界観を表現する画像を生成します。

4. **Step 2: ゲーム開始**

    - 画像が出力されたら、「ゲームマスター開始用プロンプト」をコピーして送信します。

    - AIが物語の導入と最初の選択肢を提示し、ゲームがスタートします！

## 📝 収録シナリオ

- **推理サスペンス**（ペンション殺人事件）: 吹雪で孤立した山荘で犯人を追い詰める、クローズド・サークル。

- **ダンジョンRPG（迷宮探索）**: 松明の火を頼りに、リソースを管理しながら地下迷宮からの脱出を目指す。

- **ザ・脱出（見知らぬ密室）**: 部屋に隠されたギミックを解き明かし、セキュリティを突破する王道脱出ゲーム。

- **カスタム**: あなたの想像力次第で、SFやファンタジー、恋愛シミュレーションなどあらゆる世界を構築可能。
